# 📞 Celebrity Prank Video Call

A fun and interactive **Celebrity Prank Video Call App built using Flutter**, designed to simulate realistic fake video calls with celebrities to prank friends and create entertaining moments.

> ⚠️ This app is purely for entertainment purposes only. All calls are simulated and not real.

---

## 🚀 Features

- 📹 Fake video calls with celebrities  
- 📞 Simulated incoming call screen  
- 🧑‍🎤 Multiple celebrity options  
- ⏰ Schedule fake calls  
- 🎭 Realistic UI to mimic actual video calls  
- 📸 Take screenshots and share with friends  
- ⚡ Smooth and responsive Flutter UI  

Fake call apps typically allow users to simulate calls, customize caller details, and create realistic prank scenarios for entertainment 

## 🛠️ Tech Stack

- **Flutter** – Cross-platform UI development  
- **Dart** – Programming language  
- **Material Design** – UI components  
- **Custom UI/Animations** – For realistic call experience  

---

```

lib/
````

## ⚙️ Getting Started

### Prerequisites

- Flutter SDK installed  
- Android Studio / VS Code  
- Emulator or Physical Device  

---

### 🔧 Installation

Clone the repository
```bash
git clone https://github.com/shubham230523/Celebrity_prank_video_call.git
```

Navigate to project directory
```bash
cd Celebrity_prank_video_call
```

Install dependencies
```bash
flutter pub get
```
Run the app
```bash
flutter run
```
